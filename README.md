<h3>DESCRIÇÃO DO PROJETO</h3>
Este projeto foi desenvolvido com o objetivo de praticar JavaScript e a implementação de API. A aplicação permite a busca de cep por meio de uma integração com a API e o JS, utilizando javaScript. O projeto também envolveu a estilização e formatação de botões e elementos da interface.

<h3>COMO EXECUTAR</h3>
<P>1. Baixe os arquivos do projeto.</P>
<P>2. abra o arquivo em seu navegador, insira seu CEP e clique nos botões.</P>

<h3>FUNCIONALIDADES DO PROJETO</h3>
<p>Consulta de CEP.</p>
<p>Integração com API.</p>
<p>Interface web com botões formatados.</p>
